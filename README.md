# SteamSearch

Веб-сервис для поиска игр в Steam по названию и смысловому описанию.

## Что реализовано

- Главная страница с большим полем запроса и кнопкой «Найти игру в Steam».
- Гибридный поиск:
  - **Режим A (быстрый)** — поиск по названию при коротком и "похожем на тайтл" запросе.
  - **Режим B (умный)** — для описательных запросов; используется LLM (OpenAI API) для угадывания официального названия игры.
- Выдача:
  - 1 совпадение — карточка игры, ссылка на Steam, кнопка копирования.
  - несколько совпадений — до 5 карточек с выбором.
  - нет совпадений — блок популярных игр из Steam Charts/featured categories.
- История последних 5 запросов (localStorage).
- Темная тема в стиле Steam и индикатор загрузки.
- Дисклеймер о неофициальности продукта в футере.

## Переменные окружения

Создайте `.env.local` при необходимости умного поиска:

```bash
OPENAI_API_KEY=your_key
OPENAI_MODEL=gpt-4o-mini
```

Если `OPENAI_API_KEY` не задан, режим описательного поиска деградирует к прямому поиску в Steam.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000).
